# rs-sloths

## Stack

Vue3, Pinia, Vue Router, TypeScript, Vite

## Backend
https://github.com/ThorsAngerVaNeT/rs-sloths-be

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
